# 《设计系统宪章》生成指令

请基于我提供的《项目启动大纲》，生成一份《设计系统宪章》。宪章是纯设计规则文档，不包含任何页面代码。必须包含以下章节：

### 一、色彩令牌
主色、辅色、中性色、强调色的 CSS 变量定义，暗色/浅色模式映射。

### 二、排版令牌
显示字体与正文字体的精确使用规范（字号阶梯、行高、字距、字重、标题/正文的 HTML 标签映射）。

### 三、间距与网格
基于 4px 或 8px 的全局间距标尺、栅格断点策略。

### 四、动效语法
Framer Motion 的默认过渡配置（duration、ease 曲线数组）、各类交互（悬停、点击、进入、离开）的动效参数规范。

### 五、移动端行为哲学
导航如何变形、哪些内容块会重新排序、触摸目标的最小尺寸要求。

### 六、禁则
明确列出本宪章禁止使用的样式（如圆角最大值、特定色彩组合、特定动画类型）。

---

**输出要求：**
- 输出时请使用结构化 Markdown。
- 所有数值精确到像素或 rem。
- 整份宪章将成为后续所有页面的唯一样式源。
